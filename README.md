## Features
- Introduction section
- Skills section
- Projects showcase
- Contact information

## Technologies Used

- **HTML**: For structuring the content.
- **CSS**: For styling and layout, including Flexbox and Grid for responsive design.
- **GitHub Pages**: For hosting and deploying the project for free.
- **VS Code**: My preferred code editor.

## Challenges Faced

- **Responsive Design**: Initially, I struggled with making the layout fully responsive. I had to adjust some CSS properties, such as padding and margins, to ensure sections didn’t overlap on smaller screen sizes.
- **Aligning Elements**: Ensuring perfect alignment of text and buttons was a challenge, but I solved it by using flexbox and grid layout techniques in CSS.
- **Browser Compatibility**: I encountered slight differences in how different browsers rendered the design. I used vendor prefixes to ensure cross-browser compatibility.
