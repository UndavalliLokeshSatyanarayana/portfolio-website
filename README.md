##Design Choices
- Simple and Clean Layout: I wanted to keep the design minimalistic, allowing the content to speak for itself.
- Responsive Design: The layout adjusts to various screen sizes using media queries to ensure the portfolio looks great on desktops, tablets, and mobile phones.
- Typography: Used Roboto font for clarity and professionalism, with different font weights to create emphasis.
- Color Scheme: A combination of a light background with darker text ensures readability. Accent colors are used sparingly to highlight important sections.
